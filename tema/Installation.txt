Installation

1. First of all it's required to Install Node.js 16.14 or later and npm / yarn.
2. Open your favorite console application (Terminal, iTerm, Command Prompt etc.). Navigate to a your favorite project layout folder (classic, horizontal or compact) and Install packages using following commands. This command will install all the required Node.js modules into the node_modules directory inside project folder. And now, you are ready to run the project for the first time.

npm install
or 
yarn install

3. Now you can run this Next.js project by using following command for your application development, make the necessary changes.

npm run dev
or 
yarn run dev

4.To builds the app for production you can use next build command. next build creates an optimized production build of your application. The output displays information about each route.

npm run build
or 
yarn build
